# electronjs-todo-list

[![Build Status][ico-travis]][link-travis]
[![License][ico-license]][link-license]

TODO-List based on Electron.js.

## ⚡ Installation

Via GIT

```bash
$ git clone https://github.com/voyula/electronjs-todo-list.git
$ cd electronjs-todo-list
$ npm install
```

## 🐣 Running

```bash
npm run start
```
